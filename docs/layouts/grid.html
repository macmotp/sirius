    Grid

    Flexible structure for your responsive template

    Basic

    To build a grid, just:

    • 1) Add a .grid container
    • 2) Add as many .tile elements as you want

    Tiles will always fill up the entire width of your grid. Grids can be nested: this page is a clear example.

    Responsive
    
                            
    By default, tiles are only activated from tablet onwards. This means tiles are stacked on top of each other on mobile.
    
                            
    If you want a strict grid to work on mobile too, just add the .for-mobile modifier on the .grid container:
    
                            
    
                                
    
                                    
    1
    
                                
    
                                
    
                                    
    2
    
                                
    
                                
    
                                    
    3
    
                                
    
                                
    
                                    
    4
    
                                
    
                            
    
                            
    If you only want tiles on desktop, just use the .for-desktop modifier on the .grid container:
    
                            
    
                                
    
                                    
    1
    
                                
    
                                
    
                                    
    2
    
                                
    
                                
    
                                    
    3
    
                                
    
                                
    
                                    
    4

    Tight
    
                            
    If you want a tile to only take the space it needs, use the .width-tight modifier. The other tile(s) will fill up the remaining space.
    
                            
    
                                
    
                                    
    This tile is large exactly enough to fit this text
