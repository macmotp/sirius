    Nav

    Browse responsively

    Basic

    A responsive nav bar that can contain links, tabs, buttons, icons, etc...

    The nav container has 3 parts:

    • .left
    • .center
    • .right
    Brand Name
    Link Link Button

    For mobile resolution, center and right parts will be hidden and it will appear a .toggle element. This element can always be visible adding a .visible class, like in this documentation layout.

    Try to change the resolution and see it in action.
