    Menu

    Off-canvas list

    Left or Right

    A menu is composed by a Vue template. You can use a button to trigger it from the left side of the screen or from the right.

    • From the Left
    • Link
    • Another Link
    • From the Right
    • Link
    • Another Link

    How it works? Use a custom tag sirius-menu, set a reference ref="name" and add a class .left or .right. Next you can trigger the event by adding @click="$refs.name.toggle()" to a button (or any other element).

    
                                
                            
    
                        
    
                    
    
                
    
                
    

                
    
                    
    
                        
    
                            
    Other Properties
    
                            
    There are other properties you can specify directly into your HTML:
    
                            
      
                                
    • :show-close-button: Show the close button on the top right corner (default: true)
      • 
                                
    • :close-on-external-click: Close the menu by clicking outside of it (default: true)
      • 
                            
    
                            
    Tip: remember to use the colon character before a property!

    Multi List
    
                            
    There is also a second custom template you can use inside a menu. If you have a list of links with sub-links, you can add a sirius-list tag:
    
                            
                            
    The animation is pure CSS, you won't need to edit any javascript!
    
                            
    How it works? Insert a reference name and a list-name for your sirius-list, than for your parent li element trigger the event with $refs.listName.slideIn($refs.parentListName).
    
                            
    Note: you should use the property :is-parent for your main list, then remember to pass the parent list name reference when triggering the event!
