    Image

    The right space for your pictures

    Dimensions

    Wrap your img into a .image container. In this way you will set the necessary space and the layout will not break while loading.

    If you have a square shaped image, you can specify the right dimension:

    • .dim-16x16
    • .dim-24x24
    • .dim-32x32
    • .dim-48x48
    • .dim-50x50
    • .dim-60x60
    • .dim-64x64
    • .dim-70x70
    • .dim-80x80
    • .dim-96x96
    • .dim-100x100
    • .dim-128x128
    • .dim-200x200
    • .dim-250x250
    • .dim-300x300
    • .dim-400x400

    Important: for retina displays you should use an image twice the size of the .image container

    Ratio
    
                            
    Often you do not know the size of your image. So you can specify the ratio instead:

      •

      •

      • 
                                
    • .ratio-3-2
      •
