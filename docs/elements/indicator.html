    Pagination
    
                            
    The sirius-pagination Vue template has several properties:
    
                            
      
                                
    • :list: The list of objects to paginate
      • 
                                
    • :simple: Set this to true if you want simple pagination (default is false)
      • 
                                
    • :per-page: Set this to the number of items you want to display (default is 10)
      • 
                                
    • :show-indicators: Set this to false if you want to remove the chevron indicators (default is true)
      • 
                                
    • previous-text: String to override the text of the Previous button (default is Prev)
      • 
                                
    • next-text: String to override the text of the Next button (default is Next)
      •
