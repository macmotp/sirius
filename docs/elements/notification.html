    Notification

    Uh oh. Something happened.

    Basic

    A custom template containing a message. Tag is sirius-notification.

    {{notification.message}}

    This is a Vue template, you can set :show-remove-button as property (default is true).

    Modals
    
                            
    This is another Vue template sliding from the top of the page. Tag is .sirius-modal, available in three different sizes (use small or large class to change the size):
    
                            
                            
                            
                            
                                
    
                                    
    Hello there!
    
                                
    
                                
    
                                    
    Here it is an important message for you!
    
                                
    
                                
                            
                                
    
                                    
    Here it is a less important message for you!
    
                                
    
                                
                            
                                
    
                                    
    Wow I'm big!
    
                                
    
                                
    
                                    
    I also have a footer with a button!
    
                                
    
                                
    
                                    
                                
    
                                
                            
    These are the properties you can specify directly into your HTML:
    
                            
      
                                
    • :show-close-button: Show the close button on the top right corner (default: true)
      • 
                                
    • :close-on-external-click: Close the modal by clicking outside of it (default: true)
      • 
                            
    
                            
    Be aware: remember to use the colon character before!
