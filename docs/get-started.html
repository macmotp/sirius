    Advanced Usage and Laravel Integration
    
                        
    
                            
    Sirius uses the wonderful Laravel Mix to built its assets. If you want to change SASS variables or add your own Vue components, install node dependencies:
    
                            
    
                            
    
# browse to project
cd sirius-ui
# install dependencies
npm install
                            
    
                            
    

                            
    For Laravel 5 integration, copy the assets folder into your resources project. If you are familiar with Laravel Mix, you know you can simply run
    
                            
    
                            
    
# compile assets in development
npm run dev

# compile and minify assets for production
npm run production
